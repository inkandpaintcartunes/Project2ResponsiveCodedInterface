# Project2ResponsiveCodedInterface
# Project Title
Operation Space Void
## Project Description

A game menu design.
## Intended User or Player

General audiences, with a focus on young men ages 18-24.
## User / Player Goal

The player should be able to start the game, change settings, access extras, read developer info, and quit the game.
## Inspiration Interfaces

My main inspirations were the Persona 5 and Super Smash Bros Ultimate menus. From Persona 5, I learned the importance of color palettes and stylization. Meanwhile, Smash Bros taught me how to section different elements together, especially in a manner of thirds.
## Design Choices
- Layout: I split the screen into thirds to divide the information formally.
- Color: Shades of dark blue were used throughout, as well as darker colors like black.
- Typography: For typography, I had the text be white to contrast with the majority darker colors of the menu. The font is also in-theme, with a science fiction style, called "BBB Invader."
- Visual hierarchy: The main header of the menu draws to the eye first, and then the right with , and finally the left.
- Feedback states: Uses Focus to highlight selections on the menu. 
- Responsive design: Uses Flex to abide by whatever aspect ratio it is seen in.
